Fake Format simulates the Windows format functionality. Once the program is run, no matter which buttons are chosen Fake Format starts to format the drive. The user is unable to stop, interrupt, or cancel this format. Once the fake formatting is complete, the standard Windows format summary appears. Once the user closes this summary screen the <Close> button actually closes the program.

How Do I:
  • Create a Windows shortcut?
    • Right-click on the program you want to create a shortcut for
    • Select 'Create Shortcut' from the popup menu
    • This will create a shortcut to the program at the same directory level. (See below for editing a shortcut)
  • Edit a shortcut?
    • Select/Highlight a Windows shortcut
    • Right-click on the shortcut
    • Select 'Properties' from the popup menu
    • Click on the 'Shortcut' tab
    • Modify the 'Target' field with the drive letter and path to this program name. (Ex. C:\JOKES\{program name}.EXE )
  • Edit/Add command line options?
    • Same as above except...
    • Modify the end of the 'Target' field and add your command line options

Features:
  • Un-detectable with Task Bar
  • Invisible to the CTRL-ALT-DEL (Task Manager)
  • Virtually unstoppable (ALT-F4, CTRL-Break, won't work)
