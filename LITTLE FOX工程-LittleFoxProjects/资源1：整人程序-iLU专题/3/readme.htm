Crazy Num Caps Scroll v1.04

This program is truly a jokesters dream. It will allow you to setup the toggling of the Caps, Num and Scroll lock keys.

© Copyright 1999-2000, RJL Software, Inc.


Product Description

This program will toggle the Num Lock, Caps Lock, or Scroll Lock keys at user defined intervals. It is possible to setup which keys to toggle and how often to toggle each key. This program also has command line support, so you can setup a shortcut to toggle each key. It can also be added to the Windows startup, where it would automatically begin toggling the keys. The best part about this program is that it is totally hidden to the user, and they will be unable to determine the source of the toggle.

How to Close

Move your mouse cursor to the TOP LEFT corner of the screen. A splash screen will be displayed letting you know the program was closed.

Command Line Options

For Caps Lock key - add a /c{# of seconds to toggle} to the end of the command line
For Num Lock key - add a /n{# of seconds to toggle} to the end of the command line
For Scroll Lock key - add a /s{# of seconds to toggle} to the end of the command line

Example:

"C:\MY STUFF\CRAZY_NCS.EXE" /c5 /n10 /s25 - would toggle the Caps lock key every 5 seconds, the Num lock key every 10 seconds, and the Scroll lock key every 25 seconds.
